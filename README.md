# Mahesh & Company Aspur - Agriculture E-Commerce Website

A modern, responsive e-commerce website for **Mahesh & Company**, a trusted agricultural equipment dealer located in Aspur, Rajasthan. The shop is an authorized dealer of **Sonalika Thresher** and **Honda Engines**, also providing JCB rental services and a wide range of farming tools and spare parts.

## Live Demo

Open `index.html` in any browser to view the website locally, or host on GitHub Pages for online access.

## Features

### E-Commerce Functionality
- **Product Catalog** — 24 products across 6 categories with images, ratings, and pricing
- **Shopping Cart** — Add, remove, and update quantities with a slide-out cart sidebar
- **Checkout Flow** — Complete order form with name, phone, address, and payment method (COD, UPI, Bank Transfer, EMI)
- **Order Confirmation** — Success modal with unique order ID and WhatsApp tracking link
- **Cart Persistence** — Cart data saved in localStorage, survives page refresh
- **Product Search** — Real-time search filtering by product name, category, or description
- **Category Filters** — Filter products by Threshers, Engines, JCB, Parts, Tools, or Pumps
- **Quick View** — Product detail modal with features list and WhatsApp enquiry button

### Website Sections
- **Hero Slider** — 3-slide auto-rotating banner with smooth transitions
- **Brand Strip** — Scrolling marquee highlighting dealer certifications
- **Categories** — Interactive cards linking to filtered product views
- **Products** — Full catalog with badges (Sale, New, Hot, Authorized)
- **Special Offer** — Monsoon sale banner with live countdown timer
- **Services** — JCB rental, repair, installation, delivery, buyback, EMI
- **Testimonials** — Customer reviews with star ratings
- **About Us** — Company history and feature highlights
- **Contact** — Info cards + enquiry form with subject dropdown
- **WhatsApp Chat** — Floating button for instant customer communication

### Design & UX
- Fully responsive (mobile, tablet, desktop)
- Modern green & gold color scheme (agriculture-themed)
- Scroll-reveal animations on cards and sections
- Sticky navbar with active section highlighting
- Back-to-top button

## How to Run

1. Download or clone the repository
2. Open `index.html` in any modern browser
3. No build step, no server, no dependencies required

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → select `main` → `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo-name>/`
