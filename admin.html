            How Product Sync Works

            When you add, edit, or delete products in this admin panel, the changes are saved only on your device (browser storage). To make them appear on the live website that customers visit, the products must be synced to GitHub.

            Step 1: You add/edit/delete a product in the admin panel
            Step 2: Product is saved in your browser's local storage
            Step 3: Auto-sync pushes products to GitHub (needs token below)
            Step 4: Live website loads updated products (within ~30 seconds)
            One-time setup required: You must create a GitHub token (free, takes 2 minutes) and paste it below. Without this token, products will only be saved locally and won't appear on the live website.

            Live Website Sync (GitHub) — Setup

            Connect to GitHub to automatically push product changes to the live website.

            Step-by-step: Create a GitHub Token (one-time, 2 minutes)

            1. Open GitHub token page
              Go to github.com/settings/tokens/new
              You must be logged in with the nikitasomani14 GitHub account that owns the website repo.
            2. Fill in the token details
              Note: Type a name like Admin Panel so you remember what this token is for.
              Expiration: Select No expiration so you don't have to redo this.
            3. Select permissions (scopes)
              Under "Select scopes", check only the box next to repo (Full control of private repositories). Do NOT check anything else.
            4. Generate the token
              Click the green "Generate token" button at the bottom of the page.
            5. Copy the token immediately
              You will see a token starting with ghp_. Click the copy icon next to it. Important: GitHub will only show this token once! If you leave the page, you cannot see it again.
            6. Paste token below and click Save
              Come back to this page, paste the token in the "Personal Access Token" field below, and click Save. That's it — you're done!

            After Setup — How Sync Works

            • Every time you add, edit, delete, or toggle stock of a product, it automatically syncs to the live website.
            • You can also manually sync by tapping "Update Live Website" on the Dashboard or Products tab.
            • If you see a yellow banner on the Dashboard saying "You have product changes not yet live", tap "Sync Now".
            • After syncing, it takes about 30 seconds for changes to appear on the live website.
            • The token is saved in this browser only. If you switch browsers or devices, go to Settings and paste the token again.

            Admin Guide

            Products

            Add a new product

            1. Go to the Products tab (bottom nav)
            2. Tap the + button (green circle, bottom-right)
            3. Fill in name, category, price, stock quantity, etc.
            4. To add an image: tap "Upload Photo" to pick from your device/camera, or paste an image URL
            5. Tap Save

            Edit a product

            1. Find the product in the list
            2. Tap the pencil icon on the right side
            3. Change the details and tap Save

            Delete a product

            1. Find the product in the list
            2. Tap the trash icon on the right side
            3. Confirm the deletion

            Toggle stock status

            Use the checkbox next to each product to quickly mark it as in-stock or out-of-stock without opening the edit form.

            Search & filter

            Use the search bar at the top to find products by name. Use the category chips (Threshers, Engines, etc.) to filter by category.

            Bills

            Create a new bill

            1. Go to the Bills tab, or tap "New Bill" on the Dashboard
            2. Enter the customer's name, phone, and address
            3. Add line items: enter product name (you can pick from catalog), quantity, unit price, and discount
            4. Tap "+ Add Item" to add more items
            5. Choose payment method (Cash / UPI / Bank Transfer) and status (Paid / Unpaid / Partial)
            6. Tap Save Bill

            View / Preview a bill

            1. In the Bills list, tap "View" to see the invoice preview
            2. While creating a bill, tap "Preview PDF" to see a draft

            Download PDF

            Open the invoice preview, then tap "Download PDF". The PDF file will be saved to your device.

            Share on WhatsApp

            From the invoice preview or the bill card, tap "Share" to send the bill summary via WhatsApp.

            Edit an existing bill

            In the Bills list, tap the "Edit" button on any bill card to modify it.

            Dashboard

            Overview cards

            The dashboard shows: Total Products, In Stock, Out of Stock, Low Stock (less than 3 units), and Admin Logins count.

            Website visitors

            If GoatCounter is set up, you'll see Total Visits and Unique Visitors. With an API token, you'll also see locations, devices, browsers, and referrers.

            Recent activity

            Shows the last 10 actions performed (product added/edited, bills created, etc.).

            Live Website Sync

            What is this?

            When you add, edit, or remove products in the admin panel, the changes are saved locally. Tap "Update Live Website" to push those changes to your live website so visitors can see them.

            One-time setup

            1. Go to the Settings tab (bottom nav)
            2. Find the "Live Website Sync (GitHub)" section
            3. Follow the step-by-step guide there to create a GitHub token
            4. Paste the token and click Save
            5. You only need to do this once per browser

            How to sync

            1. Make your product changes (add, edit, delete, toggle stock)
            2. Tap "Update Live Website" on the Dashboard or Products tab
            3. Wait a few seconds for the sync to complete
            4. Your live website updates automatically in about 30 seconds

            Switching browsers?

            The token is saved in your browser. If you switch browsers, go to Settings and paste the token again. You can find your existing token at github.com/settings/tokens.

            Settings

            Change password

            Enter your current password, then the new password twice, and tap Update password.

            GoatCounter Analytics

            Enter your GoatCounter site code and API token to see visitor stats on the dashboard. Follow the step-by-step guide in Settings to get your API token.

            GitHub Sync

            Enter your GitHub repository and Personal Access Token to enable one-tap product sync to the live website. See the "Live Website Sync" help section above for details.

            Clear all data

            Warning: This removes all products, bills, activity, and resets your password to the default. Use only if you want to start fresh.

            Tips
            • All data is stored on this device only (browser storage). If you clear browser data, your products and bills will be lost.
            • Use "Upload Photo" to add product images directly from your phone camera or gallery — no need to find a URL.
            • After changing products, tap "Update Live Website" to push changes to the live site. Set up a GitHub token in Settings first (one-time).
            • Tap the ? button in the top bar anytime to open this guide.
            • Log out when you're done to keep your panel secure.
